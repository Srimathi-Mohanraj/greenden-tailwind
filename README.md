🌿 GreenDen
GreenDen is a sleek and responsive web project built using HTML and Tailwind CSS, designed to showcase eco-friendly products or a nature-inspired brand. With a focus on minimalism, speed, and clean design, GreenDen offers a beautiful user interface that adapts seamlessly across devices.

✨ Features
Responsive layout using Tailwind CSS utility classes

Clean and accessible HTML structure

Smooth user experience with mobile-first design

Easy to customize and expand

This project is perfect for anyone looking to build an elegant and sustainable-looking website with modern frontend tools.
